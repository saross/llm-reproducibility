       105 +  ### Extraction System (v2.5)
       106 +  
       107 +  **Six Object Types:**
       108 +  - Evidence - Observations, measurements, data points
       109 +  - Claims - Interpretations, assertions, generalizations
       110 +  - Implicit Arguments - Unstated assumptions, logical implications
       111 +  - Research Designs - Strategic framing and rationale (WHY)
       112 +  - Methods - Tactical approaches (WHAT was done)
       113 +  - Protocols - Operational procedures (HOW specifically)
       114 +  
       115 +  **Multi-Pass Workflow:**
       116 +  - Pass 1: Liberal extraction with over-capture (comprehensive coverage)
       117 +  - Pass 2: Rationalization with consolidation (15-20% reduction)
       118 +  - Pass 3: Validation ensuring structural integrity
       119 +  
       120 +  **Provenance Tracking:**
       121 +  - Every item linked to source text via verbatim quotes
       122 +  - Location tracking (section, page, paragraph)
       123 +  - Cross-references between all object types
       124 +  - Complete consolidation metadata for traceability
       125 +  
       126 +  **Fieldwork-Specific:**
       127 +  - Respects abductive/inductive reasoning
